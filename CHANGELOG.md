## [Unreleased]

### Added

- Initial project foundation (monorepo structure, tooling, architecture)
- Frontend project scaffold with React 19, Vite, TypeScript, TailwindCSS
- Design tokens and global CSS system
- Application routing architecture
- Zustand state management stores (theme, UI)
- TanStack Query configuration
- Axios API client with interceptors
- Reusable custom hooks (useTheme, useDebounce, useLocalStorage, useMediaQuery)
- Provider architecture (QueryProvider, ThemeProvider)
- Page placeholders for all public routes
- Layout architecture (PublicLayout, BlogLayout, ToolLayout, MinimalLayout)
- Feature module placeholders
- TypeScript path aliases
- ESLint strict configuration
- Prettier formatting
- Husky Git hooks (pre-commit, commit-msg)
- Commitlint with Conventional Commits
- lint-staged automation
- Repository documentation (README, CONTRIBUTING, CODE_OF_CONDUCT, SECURITY)
- GitHub issue and PR templates
- Environment variable template
